# B64Codec-
A lightweight, portable Base64 Encoder/Decoder with a clean WinForms GUI for Windows.


## Features
- 🚀 **100% Portable:** Single executable file, less than 50KB.
- 🚫 **Zero Dependencies:** No installation required, works out of the box.
- 📋 **One-Click Copy:** Instant clipboard copy with visual feedback.
- ⌨️ **Keyboard Friendly:** Full `Ctrl+A` and shortcut support.
